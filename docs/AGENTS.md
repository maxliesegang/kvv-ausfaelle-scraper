# Published Data Agents

Guidance in this file applies to `docs/**`.
This is the most specific guidance for published artifacts.

## Output Contract

`docs/` is the published artifact directory for GitHub Pages.

- Root:
  - `docs/index.html`
  - `docs/index.json`
- Per Fahrplan year:
  - `docs/<year>/index.html`
  - `docs/<year>/index.json`
- Per line data:
  - `docs/<year>/<line>.json`
- Per year train mappings:
  - `docs/<year>/train-line-definitions/*.json`

## Data Organization Rule

Use Fahrplan years, not calendar years. Example: a cancellation on `2024-12-16` belongs to Fahrplan year `2025`.

## Change Expectations

- Most `docs/**` changes should be generated by scraper runs/workflow, not hand-edited.
- If editing manually, preserve existing JSON/HTML index structure and naming.
